
# 🚗 Rentify – Vehicle Rental Management System

**Rentify** is a web-based **Vehicle Rental Management System** developed using **Python and the Django framework** with an **SQLite database**. The system provides an online platform where users can browse available vehicles, book them for a selected date range, and manage their bookings easily.

The main objective of this project is to simplify the vehicle rental process by providing a **user-friendly online booking system**. It allows users to check vehicle availability, select booking dates, choose payment methods, and track their rental history. The admin manages vehicle details, booking records, and vehicle returns.

The application is built using **Django ORM for database management** and demonstrates features such as **authentication, role-based access control, CRUD operations, booking workflow management, payment integration, chatbot assistance, and real-time vehicle availability tracking**.

---

# ⚙️ Technologies Used

* **Backend:** Python, Django
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Database:** SQLite
* **Authentication:** Django Authentication System
* **Payment Integration:** Razorpay (Online Payment)
* **Version Control:** Git & GitHub

---

# 👥 User Roles

## User

* Register and login
* View available vehicles
* Book vehicles with selected dates
* Choose payment method (Online / Cash on Delivery)
* View booking history
* Cancel booking before the start date
* Use the **AI chatbot assistant for quick help**
* Contact **support agent through chat**

## Admin

* Add / edit / delete vehicles
* Manage vehicle details
* View all bookings
* Manage vehicle returns
* Monitor payment records
* Respond to user queries through the **chat with agent system**

# 💬 Chat Support System

The system includes two types of chat support:

* **AI Chatbot:**
  Helps users with common queries such as booking process, payment methods, and cancellation rules.

* **Chat with Agent:**
  Allows users to communicate directly with a support agent for assistance regarding bookings, payments, or vehicle issues.

This feature improves the **user experience and support system** of the application.
